    Ice Base

    Idea

    For this Blender project the idea was to create a sci-fi building in an ice environment. To make it more realistic I looked at currently existing polar stations and other buildings used in this kind of environment. The buildings should be massive to withstand the harsh weather and protect the inhabitants.

    Concept

    Based on this I created some sketches for the general look of the buildings.

    The overall layout of the base would look like this with:

    1. the central main building
    2. generators for the whole base
    3. antennas
    4. living quarters

    Modeling

    First I blocked out the basic geometry with placeholder objects:

    Compared to the additional design I added another level. The idea behind this was that the building would not be positioned directly on the ground.
